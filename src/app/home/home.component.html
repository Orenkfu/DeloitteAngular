Welcome to Oren's Deloitte-Open Legacy Project


In this angular demo, you can create, view, update and delete customers from a Java(Spring-Boot) back-end.


Instructions:

Use the navbar to navigate between the different routes in the project :)

In the Customers tab, you can view all currently existing Customers, delete them, or follow the "Update" path to navigate to the Customer Form.

In the New Customer tab, you can create a new Customer, or update an existing one if you've arrived through an "Update" link.

*Note that the only validation assigned in order to create a new customer, is that no field is empty.
